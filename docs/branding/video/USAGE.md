# Video Asset Usage

## Required videos

The hackathon requires:
- **Demo video**: 3-5 minutes, 720p minimum (1080p preferred)
- **Participant intro video**: 30-60 seconds, 16:9 landscape, front-facing

## Assets in this folder

| File | Purpose | Dimensions |
|------|---------|------------|
| `demo-title-card.svg` | Opening still for demo video | 1920x1080 |
| `intro-lower-third.svg` | Speaker overlay (transparent canvas) | 1920x1080 |

## Recommended usage

### Demo video

1. Open with `demo-title-card.svg` for 3-5 seconds
2. Fade in to live walkthrough
3. Optionally show `intro-lower-third.svg` when introducing yourself

### Participant intro video

1. Use `intro-lower-third.svg` during the first 5-10 seconds while speaking
2. Keep the background clean and the overlay in the lower-left
3. The transparent canvas allows direct compositing in any video editor

## Editing notes

Before final export, update the placeholder text if needed:
- Speaker name (currently "SP / cfxdevkit")
- Team label
- Description line

Keep the cyan left bar and dark panel — these are the strongest visual anchors in the brand system.
